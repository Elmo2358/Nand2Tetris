# Nand2Tetris
『コンピュータシステムの理論と実装第2版』のためのリポジトリ

## 概要
このリポジトリは、Nand2Tetrisプロジェクトの一環として、書籍『コンピュータシステムの理論と実装第2版』に基づいたコードとリソースを提供します。各章ごとに対応するコードを整理し、復習を容易にすることを目的としています。

## ディレクトリ構成
- `code/` - 各章ごとのコードが格納されています。
  - `01/` - 第1章のコード
  - `02/` - 第2章のコード
  - `...`
- `tools/` - Nand2Tetrisプロジェクトで使用されるツールやシミュレータが含まれています。
- `docs/` - 追加のドキュメントや参考資料。学習時のメモなど。

## 使用方法
1. リポジトリをクローンします:
    ```bash
    git clone　https://github.com/Elmo2358/Nand2Tetris.git
    ```
2. 各章のディレクトリに移動し、対応するコードを参照または実行します。
3. 必要に応じて、`tools/`ディレクトリ内のツールを使用してシミュレーションやコンパイルを行います。
